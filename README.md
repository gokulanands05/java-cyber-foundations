# Java & Cyber Security Foundations

This repository documents my continuous learning journey in
Java programming and Cyber Security fundamentals.

The focus of this repository is to build a strong foundation
by practicing concepts from the very basics and gradually
moving towards advanced levels.

All topics are practiced with small, clear examples to improve:
- Logical thinking
- Programming discipline
- Secure coding mindset
- Concept clarity

 

## Structure

- `java-basics/` 

  Contains daily Java programs starting from absolute basics
  such as printing, variables, operators, control statements,
  and object-oriented concepts.

- `cyber-basics/` 

  Contains Cyber Security fundamentals such as input validation,
  basic security principles, and secure coding concepts explained
  in a simple and practical manner.

 

## Learning Approach

- Start from zero fundamentals
- One small concept per day
- Clear and readable code
- Consistent practice
- Strong focus on understanding core principles
